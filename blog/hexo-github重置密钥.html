hexo-github重置密钥

Monkey
2021-01-29

git bash窗口依次执行

生成新秘钥：

选择文件路径，下面这个直接回车

文件已经存在，是否重写？输入 y ，回车

然后设置密码什么的，不用管，直接两下回车

新秘钥保存在 /c/Users/lenovo/.ssh/id_rsa.pub 。
打开这个文件，复制其中的的内容。
在 github 中，打开 setting -> SSH and GPG keys 。删除原来的，把新的复制进去
